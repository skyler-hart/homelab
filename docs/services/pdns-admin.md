# pdns-admin

## Overview

pdns-admin is a web-based management interface for PowerDNS. It provides zone management, record editing, and administrative controls for the authoritative DNS service.

---

## Primary Functions

- Web UI for PowerDNS zone management
- Record creation, editing, and deletion
- User and access control
- API integration with PowerDNS
- Zone file management and templates

---

## Integration

- Backend: PowerDNS API
- Frontend for zone administration
- User authentication (local or LDAP/SSO)
- Audit logging of changes
