# NetBox

## Overview

NetBox is an infrastructure asset management and Data Center Infrastructure Management (DCIM) system. It serves as the source of truth for infrastructure inventory, including devices, circuits, VLANs, and IP address space.

---

## Primary Functions

- Device and asset inventory management
- IP address management (IPAM)
- VLAN and network tracking
- Circuit and service management
- Cable and connection documentation

---

## Integration Points

- Authoritative record for all managed infrastructure
- DNS and DHCP integration (via API)
- Monitoring system integration for auto-discovery
- Backup integration

---

## Data Model

- Sites and locations
- Devices and device types
- Interfaces and connections
- IP addresses and aggregates
- VLANs and namespaces
- Circuits and services
