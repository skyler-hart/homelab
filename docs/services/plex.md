# Plex

## Overview

Plex is a proprietary media server that organizes and streams personal media collections with advanced features and cloud integration.

---

## Primary Functions

- Media library management and streaming
- Smart playlists and recommendations
- Metadata and artwork management
- DVR and live TV (with tuner)
- Remote access and sharing

---

## Features

- Automatic media discovery and organization
- Transcoding for bandwidth optimization
- Libraries and sharing policies
- Watch history and recommendations
- Family sharing and managed users

---

## Clients

- Web player
- Mobile apps
- Desktop applications
- Smart TV apps
- Hardware media players

---

## Integration

- Plex Cloud account for remote access
- Database backend
- Library storage on NAS or local volumes
- Optional: Proxied through Traefik for LAN access
- Backup of Plex configuration and database
